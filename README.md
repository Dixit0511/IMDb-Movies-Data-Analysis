# IMDb Movies Data Analysis Dashboard

## Project Overview
This project analyzes IMDb movie data using Microsoft Excel. The dataset was cleaned, transformed, and analyzed to identify trends and generate meaningful insights.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Slicers

## Dataset Information
- Total Records: 100
- Total Columns: 10

## Analysis Performed
- Genre-wise Average IMDb Rating
- Director-wise Movie Count
- Language-wise Movie Distribution
- Year-wise Movie Releases
- Top 10 Movies by IMDb Rating

## Dashboard Features
- Interactive Filters (Genre, Language, Release Year)
- Dynamic Charts
- Easy-to-understand Visualizations
